## 出力フォーマット

改善提案を以下の JSON 形式で出力してください:
```json
{"proposals": [{"title": "改善タイトル", "description": "具体的な改善内容", "priority": "high/medium/low"}]}
```
提案がなければ空配列 `{"proposals": []}` を返してください。
最大3件まで。最も効果的な改善のみ提案してください。
